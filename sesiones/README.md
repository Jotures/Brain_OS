# 📂 Sesiones de Estudio

Registro automático de tus sesiones de estudio con Brain OS.

---

## Estructura

```
sesiones/
├── 2026-02/                    # Mes actual
│   ├── 2026-02-04_sesion.md    # Sesión del día
│   ├── 2026-02-05_sesion.md
│   └── ...
└── resumen_semanal.md          # Análisis semanal
```

---

## Formato de Sesión

Cada archivo de sesión registra:
- ⏱️ Pomodoros completados
- 📚 Cursos estudiados
- 📝 Notas rápidas
- ❓ Dudas pendientes
- 🎯 Progreso hacia metas

---

## Uso

Brain OS crea automáticamente el log cuando dices:
- "Registra mi sesión"
- Al finalizar pomodoros
- Con el comando `/brain-os-study`
