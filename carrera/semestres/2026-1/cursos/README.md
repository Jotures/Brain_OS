## Estructura por Curso

```
📂 curso/
├── 📄 README.md        # Info del curso, sílabo, profesor
├── 📂 notas/           # Apuntes de clase y estudio
├── 📂 tareas/          # Tareas y entregables
├── 📂 examenes/        # Preparación para exámenes
└── 📂 recursos/        # PDFs, materiales, links
```

---

## Uso

1. **Durante clase**: Guarda notas en `notas/`
2. **Tarea asignada**: Crea archivo en `tareas/`
3. **Pre-examen**: Usa `examenes/` para repasos
4. **Materiales**: PDFs y recursos en `recursos/`
